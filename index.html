POST /api/votes Voting Platform

A secure online voting system where every registered voter can cast exactly one vote per election — no duplicates, no unauthorized submissions, no tampering with results after the fact.

ProblemElections need to guarantee one-person-one-vote without letting admins or outsiders alter outcomes.
ApproachRole-based authorization (voter / admin) with JWT, so vote-casting routes are only reachable by verified, unvoted users; bcrypt-hashed credentials protect accounts.
Data modelSeparate schemas for candidates, elections, and votes, with a vote record tied to a unique user–election pair to enforce the single-vote rule at the database level.
ResultFully tested REST API (Postman) for candidates, elections, and voting, deployed live on Render.
